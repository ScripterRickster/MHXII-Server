FAQ

General

Use the Pi URL field to connect the dashboard to your Raspberry Pi. When the Pi is connected, the live feed, trash list, and robot controls update automatically.

Technical

API requests are plain HTTP requests to the Flask server. Use the endpoints below from your Pi, a script, or another client.

POST /feed/upload
Content-Type: application/json
{
  "image": "data:image/png;base64,..."
}
POST /locations
Content-Type: application/json
{
  "lat": 43.16,
  "lon": -79.24,
  "device": "pi-1"
}
GET /robot/status
POST /robot/toggle
POST /robot/set-pi-url
Content-Type: application/json
{
  "pi_url": "http://192.168.1.100:5000"
}

If you are calling from a script, use the Render service URL as the base, for example https://your-service.onrender.com.
